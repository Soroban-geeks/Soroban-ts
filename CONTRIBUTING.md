# Contributing to soroban-ts

Thank you for your interest in contributing.

## Setup
```bash
git clone https://github.com/soroban-ts/soroban-ts.git
cd soroban-ts
npm install
```

## Development
```bash
npm run dev       # watch mode
npm test          # run test suite (68 tests)
npm run typecheck # TypeScript check
npm run build     # compile to dist/
```

## Pull Request Guidelines
- All PRs must pass `npm test` and `npm run typecheck`
- Match the existing code style
- Add tests for any new behaviour
- Keep PRs focused — one feature or fix per PR

## Issue Labels
Issues tagged `Stellar Wave` are part of the Drips Wave program.
See complexity levels: Trivial / Medium / High.
